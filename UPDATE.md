# Инструкция по обновлению

## 1. Обновите код с GitHub

```bash
cd /Users/timmy/Desktop/fish
git pull origin main
```

## 2. Полностью закройте приложение

Если приложение запущено - закройте его полностью (не просто окно, а весь процесс).

## 3. Запустите приложение заново

```bash
python3 main.py
# или
python main.py
```

- ✅ Кнопка "Добавить в лог" - добавление рыбы

### Если кнопки всё ещё не работают:

1. Проверьте консоль/терминал на ошибки
2. Убедитесь, что используется правильная версия Flet (0.24.0+)
3. Попробуйте удалить кеш Python: `find . -type d -name "__pycache__" -exec rm -r {} +`
